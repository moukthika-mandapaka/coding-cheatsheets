HTML Cheatsheet

This page covers common HTML tags, attributes, and elements (based on Stanford’s HTML reference).

Basic Structure


<!DOCTYPE html>
<html>
  <head>
    <title>Title of the document</title>
  </head>
  <body>
    Content goes here
  </body>
</html>

Text Formatting

  • <b> – Bold text
  • <i> – Italic text
  • <u> – Underlined text
  • <em> – Emphasized text
  • <strong> – Important text (bold)
  • <br> – Line break
  • <hr> – Horizontal rule
  • <sub> / <sup> – Subscript / Superscript

Headings


<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

Lists

  • <ul> – Unordered list
  • <ol> – Ordered list
  • <li> – List item

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

Links & Images

  • <a href="url">link</a> – Creates a hyperlink
  • <img src="image.jpg" alt="desc"> – Adds an image

Tables


<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>

Forms


<form action="submit.php" method="post">
  <label>Name:</label>
  <input type="text" name="username">
  <input type="submit" value="Submit">
</form>

Common Tags

  • <div> – Container for content
  • <span> – Inline container
  • <p> – Paragraph
  • <blockquote> – Quotation block
  • <pre> – Preformatted text
