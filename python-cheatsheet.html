Python Cheatsheet

Quick look at common Python syntax and operations.

Variables & Data Types

x = 5
name = "Krina"
pi = 3.14
is_valid = True

Conditionals

if x > 10:
    print("Big")
elif x == 10:
    print("Equal")
else:
    print("Small")

Loops

for i in range(5):
    print(i)

while x > 0:
    x -= 1

Functions

def greet(name):
    return f"Hello {name}"

print(greet("Katherine"))
